Sky Leaper

After graduating high school, four friends and I were interested in devloping a mobile app. After some brainstorming, we decided we wanted to make a game and were tasked with developing an enjoyable game with a professional look. We decided the best way to accomplish this was using the the Unity Physics Engine. In addition, we used GitHub for version control and had frequent scrums to create the game. By the end of the three months, we had a finished project that we uploaded to the Google Play Store. While working on this project I learned how to efficiently develop software in groups. I also learned more about the software development process as our expectations of the game changed as we worked on the game.
